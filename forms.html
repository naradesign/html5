datalist element

input list="local-list" + datalist id="local-list"

output element

form oninput="x.value=parseInt(a.value)+parseInt(b.value)" + = 108
form oninput="y.value=parseInt(c.value)+parseInt(d.value)" + = 100

progress element

progress value="0.5" 0.5/1
50/100

min 속성은 존재하지 않음. max 속성 생략시 기본 값은 1.

meter element

meter value="0.5" 0.5/1
50/100

min/max 속성 생략시 기본 값은 0/1이 된다. 'low, high, optimum' 속성을 사용하면 '낮은 범위, 높은 범위, 적정 값'을 명시할 수 있음.

input [type] attributes

input type="search"
input type="tel" - -

전화번호를 입력해 주세요. 예) 010-3405-9380

input type="url"

웹 사이트 주소를 입력해 주세요. 예) http://naradesign.net/

input type="email"

이메일 주소를 입력해 주세요. 예) email@email.com

input type="number"

번호(정수)를 입력해 주세요. 예) 0123456789, -0123456789

input type="date"

날짜를 규격에 맞게 입력해 주세요. 예) 2010-12-24

input type="time"

시간을 규격에 맞게 입력해 주세요. 예) 00:00

input type="range"

수준을 입력해 주세요. 예) 0 ~ 100

input type="color"

색상을 입력해 주세요. 예) #FF0000, #ff0000, RED, red

input type="file" accept="image/*"

하나 이상의 이미지 파일을 첨부해 주세요.

input attributes

input required

이메일 주소는 반드시 입력해야 합니다.

input autocomplete="on"
input autocomplete="off"
input autofocus="autofocus"
input placeholder="abc@abc.com"
input list="local-list" + datalist id="local-list"
input min="0" max="10"

숫자 최솟값은 '0' 이상, 최댓값은 '10' 이하여야 합니다.

input minlength="4" maxlength="8"

문자 길이 최솟값은 '4' 이상, 최댓값은 '8' 이하여야 합니다.

input step="10"

0~100 가운데 10 단위로만 입력하세요.

input type="email" multiple

콤마 ','로 분리하여 여러 개의 이메일 주소를 입력할 수 있습니다.

input type="file" multiple

여러 개의 파일을 첨부할 수 있습니다.

select multiple

여러 개의 값을 선택할 수 있습니다.

input pattern="[0-9][0-9][0-1][0-9][0-3][0-9]-[1-4][0-9][0-9][0-9][0-9][0-9][0-9]"

가상의 주민등록번호를 입력 해보세요. 예) 000000-1000000
