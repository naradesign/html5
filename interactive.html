HTML5 interactions

details/summary element

details 요소란?

정보나 콘트롤을 드러내는 위젯이다. details 요소에 open 속성을 넣으면 열린 상태로 표시한다.

summary 요소는 details 요소의 나머지 부분에 대한 요약, 캡션, 범례를 의미한다.

숨겨진 폼(form)을 드러내기도 한다.

menu/menuitem element

파이어폭스에서 이 영역에 마우스 포인터를 올려놓고 우측 버튼을 누르면 문맥 메뉴가 표시 됨. menu 요소는 명령 그룹을 의미한다. menuitem 요소는 팝업메뉴(문맥메뉴)에서 사용자가 작동할 수 있는 명령을 의미한다.(2007.3 현재 파이어폭스만 지원.)
